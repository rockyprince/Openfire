APNS Plugin Readme

Overview

The APNS plugin is a service that provides push notification to iOS devices.

Installation

Copy apns.jar into the plugins directory of your Openfire installation. The plugin will then be automatically deployed. To upgrade to a new version, copy the new apns.jar file over the existing file.

Configuration

The apns plugin can be configured via the Openfire Admin Console.

Using the Plugin

Just use it directly.
